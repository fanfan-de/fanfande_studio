## OpenCode 项目架构分析

这是一个 **AI 驱动的代码编辑代理** 项目，采用 **Monorepo** 结构（使用 Bun 工作区管理）。以下是详细的架构分析：

### 📱 **前端应用**

#### 1. **Web 应用 ([web](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html))**

- 框架：Astro + SolidJS
- 用途：官方营销网站和文档站点
- 域名：opencode.ai

#### 2. **Console 应用 ([console](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html))**

- 主应用前端
- 框架：SolidJS + Vite
- 包含子目录：[app/](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html)、`core/`、`function/`、`mail/`、`resource/`
- 用途：用户管理界面、项目管理、AI 代理交互

#### 3. **Desktop 应用 ([desktop](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html))**

- 框架：Tauri（桌面应用框架）+ Vite + SolidJS
- 复用：[@opencode-ai/app](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html) 和 `@opencode-ai/ui`
- 平台支持：macOS (ARM64/x64)、Windows、Linux (.deb/.rpm/AppImage)
- 功能：本地桌面版本的 OpenCode

#### 4. **OpenCode CLI ([opencode](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html))**

- 框架：Node.js CLI 工具
- 用途：命令行交互式 AI 代理（支持 Terminal UI）
- 发行：NPM、Homebrew、Scoop、Arch、Nix 等多种包管理器

#### 5. **Enterprise SaaS 平台 ([enterprise](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html))**

- 框架：SolidStart（Solid.js 的全栈框架）
- 部署：Cloudflare Workers + R2 存储
- 用途：企业版多团队协作平台

### 🔧 **共享前端资源**

#### **UI 组件库 ([ui](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html))**

- SolidJS 组件库
- 包含：
    - UI 组件、主题、i18n 国际化
    - 图标库（provider icons、file type icons）
    - Pierre 差异展示库
    - Tailwind CSS 样式

#### **应用基础库 ([app](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html))**

- 框架：Vite + SolidJS
- 用途：Web 和 Desktop 应用的共享代码基础

### 🖥️ **后端服务**

#### 1. **API Worker ([api.ts](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html))**

- 框架：**Cloudflare Worker** + Hono
- 部署：Cloudflare Workers
- 域名：[api.{domain}](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html)
- 功能：
    - GitHub App 集成（App ID、Private Key）
    - Admin 管理
    - Discord 支持机器人
    - Feishu（飞书）集成
    - Bucket 存储管理
    - Durable Objects（SyncServer）用于实时同步

#### 2. **数据库**

- **PlanetScale**（MySQL 兼容）
- 环境：
    - 生产环境：main 分支
    - 开发环境：动态分支（每个 stage 一个）
- 管理工具：Studio（可视化管理）

#### 3. **存储**

- **Cloudflare R2**（S3 兼容对象存储）
- 用于企业平台文件存储

- [slack](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html) - Slack 集成

### 🏗️ **基础设施（使用 SST）**

配置文件：[sst.config.ts](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html)、[infra](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html)

- **云平台**：Cloudflare（Workers、R2、KV）
- **数据库**：PlanetScale
- **支付**：Stripe
- **多阶段部署**：development、staging、production

### 💾 **项目配置**

- **包管理**：Bun v1.3.5
- **语言**：TypeScript
- **构建工具**：Turbo (单体仓库编排)
- **样式**：Tailwind CSS v4
- **代码质量**：Husky (git hooks)

这是一个现代化的全栈 AI 工具项目，采用了 **边缘计算**（Cloudflare Workers）+ **数据库**（PlanetScale）+ **多前端**（Web/Desktop/CLI）的架构模式。
