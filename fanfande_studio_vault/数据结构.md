### 1. **Agent.Info** 
| 字段名               | 类型 (TypeScript / Zod)    | 约束/备注                                   | 是否必填  |
| :---------------- | :----------------------- | :-------------------------------------- | :---- |
| **`name`**        | `string`                 | Agent 的名称                               | **是** |
| **`description`** | `string`                 | Agent 的功能描述                             | 否     |
| **`mode`**        | `enum`                   | 可选值: `"subagent"`, `"primary"`, `"all"` | **是** |
| **`native`**      | `boolean`                | 是否为系统内置 Agent                           | 否     |
| **`hidden`**      | `boolean`                | 是否在列表中隐藏                                | 否     |
| **`topP`**        | `number`                 | 核采样参数                                   | 否     |
| **`temperature`** | `number`                 | 温度参数（随机性控制）                             | 否     |
| **`color`**       | `string`                 | 界面显示的颜色（Hex 或名称）                        | 否     |
| **`permission`**  | `PermissionNext.Ruleset` | 权限规则集，控制文件/网络访问                         | **是** |
| **`model`**       | `object`                 | 包含 `modelID` 和 `providerID` (均位 string) | 否     |
| **`prompt`**      | `string`                 | 系统提示词 (System Prompt)                   | 否     |
| **`options`**     | `Record<string, any>`    | 动态配置键值对映射表                              | **是** |
| **`steps`**       | `number`                 | 必须为**正整数** (`int().positive()`)         | 否     |
