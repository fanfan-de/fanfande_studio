- **边界（worktree / sandboxes）**: [worktree](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html) 是项目的顶层工作目录（git common dir），[sandboxes](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html) 是该 project 下的多个子工作目录 / worktrees。Agent 通过 Instance 提供的 [worktree](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html) 与 [directory](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html) 来判断工作边界。
- **VCS 信息**: 如果为 git，Project 会把 [vcs](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html) 标为 ["git"](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html)，并通过 [vcs.ts](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html) 公开当前分支与分支变更事件供 agent 使用。
- **元数据**: [name](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html), [icon](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html), [commands.start](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html), [time](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html)（创建/更新时间/initialized）等可被更新并通过事件广播，agent 可读取并展示或依赖这些元信息。

**Instance（上下文）与 agent 的关系**

- Agent 在某个目录运行时应调用 [Instance.provide({ directory, fn })](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html) 来：
    - 确定对应 [Project.fromDirectory(directory)](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html)（即决定 project id/worktree/sandbox）；
    - 设置上下文，使后续库调用（VCS、File、Format、Plugin 等）基于该 instance 工作。
- [Instance.containsPath(filepath)](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html) 用于判断文件是否属于当前 project（或其 worktree），这是决定是否允许访问/标记为 external 的关键（权限边界）。
- [Instance.state(...)](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html) 提供每-instance 的局部长期状态（例如 vcs state），agent 在生命周期内可以重用这些状态；[Instance.dispose()](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html)/[disposeAll()](vscode-file://vscode-app/e:/Microsoft%20VS%20Code/resources/app/out/vs/code/electron-browser/workbench/workbench.html) 负责清理。
