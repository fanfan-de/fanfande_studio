# 📘 VS Code IntelliSense 符号图标全解析

这些图标出现在 **代码补全列表 (IntelliSense)**、**大纲视图 (Outline)**、**面包屑导航 (Breadcrumbs)** 以及 **搜索符号 (@ 符号)** 中。

## 1. 核心图标对照表

根据图标代表的代码逻辑，可以分为以下四大类：

### 🔧 行为与执行 (紫色系)
这类图标通常表示“可以执行的代码段”。

| 图标           | 名称                    | 描述 / 示例                                      |
| :----------- | :-------------------- | :------------------------------------------- |
| 🟣 **立方体**   | **Method / Function** | 类的**方法**或普通**函数**（如你图中的 `load`, `loadFile`）。 |
| 🟣 **方块+括号** | **Constructor**       | 类的**构造函数**。                                  |

### 📦 数据与属性 (蓝色/青色系)
这类图标通常表示“存储的数据”或“成员定义”。

| 图标 | 名称 | 描述 / 示例 |
| :--- | :--- | :--- |
| 🟦 **实心/空心方块** | **Property / Field** | 对象的**属性**、**字段**或**变量**（如你图中的 `Server`, `Info`）。 |
| `{ }` **大括号** | **Module / Object** | **模块**、**命名空间**或 JSON **对象**（如你图中的 `Config`）。 |
| 🔲 **圆圈/点** | **Variable** | 局部或全局**变量**。 |
| 🧊 **蓝色立方体** | **Class** | **类**的定义。 |

### 接口与定义 (黄色/橙色系)
这类图标通常表示“代码的结构或契约”。

| 图标           | 名称            | 描述 / 示例                 |
| :----------- | :------------ | :---------------------- |
| 🧱 **台阶状图标** | **Interface** | **接口**。在 Go 或 TS 中非常常见。 |
| 📜 **枚举图标**  | **Enum**      | **枚举类型**。               |
| 📄 **常量图标**  | **Constant**  | **常量**，通常不可修改。          |

### 🛠 其他辅助类
| 图标 | 名称 | 描述 |
| :--- | :--- | :--- |
| 💬 **气泡** | **Text** | 纯文本建议（通常来自当前文件的分词）。 |
| ✂️ **剪刀/标签** | **Snippet** | 代码**片段**（常用代码模板）。 |
| 🔑 **钥匙** | **Keyword** | 语言**关键字**（如 `if`, `return`, `export`）。 |
| 📂 **文件夹** | **Folder** | 路径补全中的目录。 |

## 2. 针对你截图中图标的具体解析

结合你提供的截图（可能是一个配置文件或 Go/TypeScript 的结构定义）：

1.  **`{ }` Config**: 这是一个顶层的**对象 (Object)** 或 **模块 (Module)**。
2.  **🟦 Server / Layout / Info**: 这里的蓝色方块图标代表**属性 (Property)**。
    *   *提示：* 带 `>` 箭头的表示该属性是一个复杂对象，可以展开查看其内部成员；不带箭头的表示它是一个具体的值或基本类型。
3.  **🟣 loadFile / load**: 紫色立方体代表**方法 (Method)** 或 **函数 (Function)**，表示这些是可以被调用的逻辑。
4.  **🟦 global**: 通常是定义的全局**属性**或变量。

## 3. 如何快速确认图标含义？

如果你以后遇到不确定的图标，有三个方法可以快速查看：

*   **方法 A (IntelliSense 提示):** 在代码中触发补全（`Ctrl + Space`），选中该项，右侧通常会弹出详细的类型说明（如 `property`, `method`）。
*   **方法 B (官方图标库):** 访问 [VS Code Product Icon Reference](https://code.visualstudio.com/api/references/icons-in-labels#icon-listing) 搜索 `symbol-` 开头的 ID。
*   **方法 C (主题相关):** 
    *   图标的具体形状和颜色可能会随你安装的 **Product Icon Theme** 而改变。
    *   如果你想换一套更直观的图标，可以在扩展商店搜索 `Material Product Icons`。

---
> **💡 小技巧：** 在 VS Code 中按下 `Ctrl + Shift + O` (或 Mac 上的 `Cmd + Shift + O`)，输入 `:`，你会看到所有符号按类别分组，组名会直接告诉你图标的含义。
