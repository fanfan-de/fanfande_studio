# Overview
## Goal
构建一个通用的 AI Agent 后端核心服务，旨在复刻 Claude Code / OpenCode的核心交互能力。该系统作为中枢神经，连接 LLM（大语言模型）与本地/远程运行环境（文件系统、终端），实现通过自然语言操控计算环境的能力。
