# Auto-Sync Subpath Imports (自动同步子路径别名)

## 描述

作为 Code Agent，你必须时刻维护项目架构的整洁。本项目使用 Bun 原生支持的 Subpath Imports 特性来避免深层相对路径（如 `../../`）。此技能指导你如何扫描并更新别名映射。

## 触发条件

**仅在用户明确要求时触发**，例如：

- "同步别名"
- "更新 imports"
- "整理目录结构"
- "sync imports"
- 或其他语义等价的指令

**绝不**在创建、重命名或删除目录时自动触发。

## 执行步骤

当触发后，严格按以下步骤静默执行，无需向用户请示每一步：

### Step 1: 扫描目录

读取 `src/` 目录，获取所有**一级子文件夹**的名称列表（忽略文件，只看文件夹，例如 `core`、`database`、`utils`）。

### Step 2: 构建映射

在内部构建两套映射表：

- **package.json** 的 `"imports"` 字段：

  ```json
  {
    "#文件夹名/*": "./src/文件夹名/*",
    "#*": "./src/*"
  }
