# 桌面端界面结构与命名

最后更新: 2026-04-09  
适用范围: `packages/desktop` / `src/renderer/src`  
说明: 本文档从 `AI_AGENT_FRONTEND_SPEC.md` 中拆出，专门维护稳定界面结构、命名和测试依赖的定位锚点。

## 1. 当前界面结构

```text
Window Shell
├─ Titlebar
└─ App Shell
   ├─ Activity Rail (optional, left only)
   ├─ Left Sidebar (optional)
   │  ├─ Sidebar Actions
   │  ├─ Folder Workspace List
   │  │  └─ Project Block
   │  │     ├─ Folder Row
   │  │     └─ Session Tree (expanded only)
   │  └─ Settings Entry
   ├─ Left Sidebar Resizer (optional)
   ├─ Canvas
   │  ├─ Canvas Top Menu
   │  ├─ Thread Shell
   │  │  ├─ Empty State Assistant Turn
   │  │  ├─ User Turn
   │  │  ├─ Assistant Turn
   │  │  │  └─ Assistant Trace List
   │  │  └─ Permission Request Inline Prompt (optional)
   │  └─ Composer
   │     ├─ Attachment Strip (optional)
   │     └─ Composer Toolbar
   │        ├─ Model Menu
   │        └─ Agent Mode Menu
   ├─ Right Sidebar Resizer (optional)
   ├─ Right Sidebar / Inspector (optional)
   └─ Settings Page Overlay (optional)
      └─ Settings Page
         ├─ Settings Header
         └─ Settings Page Shell
            ├─ Settings Primary Nav
            └─ Settings Page Main
               ├─ Provider Layout
               ├─ Models Layout
               └─ Appearance Layout
```

结构补充说明：

1. `Activity Rail` 只存在左侧，没有右侧对称 rail。
2. `Right Sidebar` 的定位是 Inspector，不是 `Canvas Tool Panel`。
3. `Settings Page Overlay` 是 `App Shell` 的覆盖层 sibling，不是 `Canvas` 的子区域。
4. `Canvas Top Menu` 当前不驱动内容切换，只承载静态菜单项和两侧边栏的恢复入口。
5. 常规 assistant turn 在 UI 上只显示非 `system` trace item；待审批卡片是 thread 内的独立区域。

## 3. 术语约束

1. 左侧导航统一称“文件夹工作区”“workspace”“session”，不要混用 `project tree`、`folder tree` 指代不同层次。
2. 右侧栏统一称 `Right Sidebar` 或 `Inspector`，不要写成 `Canvas Tool Panel`。
3. 设置页一级分组统一使用 `Provider`、`Models`、`Appearance`，不要沿用旧的 `Model Services`、`Default Models`。
4. 左侧窄轨统一称 `Activity Rail` 或“左侧窄轨”，不要和左侧栏混称。

## 4. 维护规则

1. 新增稳定界面区域、ARIA label 或测试锚点时，先更新本文档，再改代码。
2. 编写测试时，优先使用本文档中的类名、ARIA label 或稳定文案定位。
3. 如果右侧 Inspector、Canvas Top Menu 或 Settings Page 的结构职责变化，必须同步更新本文档和 `AI_AGENT_FRONTEND_SPEC.md`。
