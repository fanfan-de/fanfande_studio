# FAQ

## Anybox 是什么？

Anybox 是面向本地项目工作的 AI Agent 桌面工作台。仓库里还包含本地 Agent 服务、桌面端和配套站点。

## 当前支持哪些平台？

当前网站提供 Windows x64 EA 安装包和 macOS Apple Silicon 安装包。Linux 版本还在开发中。

## 是否必须连接云端模型？

不一定。Anybox 的目标是支持多供应商和灵活配置。你可以连接云端模型，也可以按项目部署方式连接自定义 Agent 服务。

## 会不会自动修改我的项目？

Anybox 围绕本地项目工作。涉及文件修改、终端命令和工具调用时，应通过会话界面查看变更，并在需要审批的操作上明确确认。

## 下载链接失效怎么办？

如果首页或文档页下载失败，可以直接打开 GitHub Releases，选择最新 Windows x64 或 macOS Apple Silicon 安装包。

## 文档还会补充哪些内容？

后续适合继续补充：详细设置项、MCP 服务配置、权限审批说明、终端面板、会话导出、团队协作和发布流程。
