# 快速开始

Anybox 是面向本地项目工作的 AI Agent 桌面工作台。你可以用它打开一个项目目录，创建会话，让 Agent 在当前工作区里阅读代码、执行任务、调用工具并跟踪变更。

## 安装

当前网站提供 Windows x64 安装包。点击首页或文档页右上角的 `win下载`，会跳转到最新发布版本的安装包。

```text
平台状态
Windows x64: EA 版本
macOS: 开发中
Linux: 开发中
```

安装完成后启动 Anybox。如果系统提示安全确认，请确认安装包来自项目的 GitHub Releases 页面。

## 第一次启动

1. 打开 Anybox。
2. 选择一个本地项目文件夹作为工作区。
3. 在设置中配置模型供应商和模型。
4. 新建会话，输入你希望 Agent 完成的任务。

## 基本工作流

Anybox 的核心工作流围绕项目目录和会话展开：

- 工作区对应一个本地项目目录。
- 会话保存一次 Agent 协作过程。
- 终端、工具调用、补丁和错误会在会话中呈现。
- 涉及高风险操作时，应用会通过权限审批链路让你确认。

## 下一步

完成安装后，继续阅读模型供应商配置。模型可用之后，再为常见工作沉淀 Skills，让 Agent 更稳定地遵循你的项目习惯。
