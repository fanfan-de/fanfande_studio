# Spreadsheets

Spreadsheets is an Anybox plugin for spreadsheet artifact workflows.

It bundles a spreadsheet-focused Agent skill that helps create, edit, analyze,
visualize, render, and export workbook files locally.

## Included Skills

- `Spreadsheets`: create, edit, analyze, visualize, render, and export spreadsheet files such as `.xlsx`, `.xls`, `.csv`, and `.tsv`.

## Discoverability

Use this plugin for spreadsheet-oriented terms from the file-type naming model: sheet, sheets, Excel, CSV, model, spreadsheet, spreadsheets, workbook, tracker, and `.xlsx`.

## Source And Copy Flow

This plugin was migrated into the Anybox third-party plugin format from the
