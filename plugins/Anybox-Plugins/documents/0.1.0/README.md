# Documents

Documents is an Anybox plugin for DOCX document artifact workflows.

It bundles a document-focused Agent skill that helps create, edit, redline,
comment on, render, verify, and export Word document files locally.

## Included Skills

- `Documents`: create, edit, inspect, render, redline, comment on, verify, and export `.docx` documents.

## Discoverability

Use this plugin for document-oriented terms from the file-type naming model:
doc, docs, Word, memo, write, document, documents, report, writing, and `.docx`.

## Source And Copy Flow

This plugin was migrated into the Anybox third-party plugin format from the
MIT-licensed Documents runtime package.
