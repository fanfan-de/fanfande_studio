# 一. 前言

## 1.1 主流手柄介绍

游戏手柄作为玩家与游戏之间的桥梁，让玩家能够精准地操控按键与游戏世界进行互动，它的重要性不言而喻。目前， 市面上主流的游戏手柄包括Xbox手柄、PS手柄和NS手柄，这3款手柄在设计上有一些区别，主要体现在部分按键的布局和名称上。手柄的按键布局决定了它适合玩哪类游戏，进而决定了玩家将获得何种游戏体验。

![](http://km.fp.ps.netease.com/file/652e6eef1abd9cc6dab14fc6r3coQPam05)

Xbox手柄采用了经典的非对称布局，握持手感非常出色。摇杆位置设计合理，使得左右手的拇指都能轻松适用，而且即使长时间操作也不会造成虎口酸痛。Xbos手柄的按键设计偏硬，更适合玩短平快的车枪球类游戏。

PS手柄采用了独特的对称布局，方向键可以很轻松地被左手拇指驾驭，因此之前很长一段时间被认为是玩横板卷轴游戏的最佳选择。PS手柄按键较软且键程短，也很适合玩音游之类的爆手速游戏。除此之外，PS手柄的按键布局导致“C手”操作成为可能，极大提高了动作格斗类游戏中玩家的操作上限。但是，在使用PS手柄时，拇指和食指开合度加大，虎口长期拉伸，比较容易出现手指酸痛问题。

至于Nintendo Switch手柄，它的设计是能够适应多种游戏场景的可拆卸手柄。它可以作为整体手柄使用，也可以将两侧分别使用，每个侧面都有一个摇杆和一些基本的功能按钮。这种设计使得Nintendo Switch手柄非常灵活和便携。

随着近年来各平台游戏类型覆盖面越来越广，曾经主要玩车枪球游戏的Xbox手柄和主要玩日系RPG和横板卷轴游戏的PS手柄都越来越全能。

## 1.2 手柄按键热区（以Xbox手柄为例）

# 二. 案例分析

本文案例主要选自三款大逃杀射击游戏，其中包括传统的大逃杀游戏《PUBG》，科幻元素的大逃杀游戏《APEX》，以及军事背景的大逃杀游戏《CODM 战区2.0》。此外，还选取了两款掩体射击游戏，包括现代军事题材的TPS《全境封锁1》和科幻主题的TPS游戏《Outriders》。最后，还包括一款开放世界TPS游戏《幽灵行动：断点》。这些游戏代表了不同类型和风格的射击游戏，为进行案例分析提供了丰富的素材。

## 2.1 角色行动操控

任何一款3D射击游戏中，角色行动的操控都是最基础和最重要的功能，操控的流畅性和准确性直接影响了玩家的游戏体验。大逃杀射击游戏中，角色移动、奔跑、跳跃、下蹲、卧倒均属于基础行动。掩体射击游戏一般不具有跳跃、下蹲和卧倒等角色行动，取而代之的是翻滚、进入掩体和翻越掩体。具体键位设置见下图：（![](http://km.fp.ps.netease.com/file/65220f10bc0de51387fa7b9dLxfyom9905?fop=imageView/6/f/webp/q/50)表示按住，![](http://km.fp.ps.netease.com/file/652118e1b09ad29ccc3c1ecdooM5Jopo05?fop=imageView/6/f/webp/q/50)表示双击）

![](http://km.fp.ps.netease.com/file/652f5550ffd7b740a006af3dWQWPcQ2s05?fop=imageView/6/f/webp/q/50)

从上图各游戏中角色基础行动的按键设置，以及结合手柄右摇杆调整视角的设定，不难看出手柄的L键、R键、A键和B键是最高频使用的按键。其他说明如下：

1. 一般情况下角色**移动、奔跑、跳跃、下蹲、卧倒、自动奔跑**等功能可以采用图示进行按键设置，基本可以符合大多数玩家的操作习惯。

![](http://km.fp.ps.netease.com/file/652f55a5ffd7b740a006b193U4MiaM3E05?fop=imageView/6/f/webp/q/50)

2. 虽然按下L键触发**奔跑**的设定很常见，但还是被不少玩家所吐槽，原因是持续按下摇杆不仅影响操控精度而且容易造成手指疲劳。因此，大多数游戏均会在操作设置中允许玩家将“按住触发”改为“点击触发”。也有不少游戏通过组合键来触发奔跑，比如《The Last of Us》中向前推动L键时按下LB键触发奔跑，《Gears 5》向前推动L键时按下A键触发奔跑。  
  
3. **自动奔跑**大多采用双击L键来实现，也有部分游戏向前推动L键触发自动奔跑。《全境封锁1》使用键鼠输入时点击键盘X键实现自动奔跑，但是手柄输入时无此项功能，不过在《全境封锁2》中使用手柄输入时，双击L键仍是可以控制角色自动奔跑的。  
  
3. 掩体交互相关的操作主要有**进入掩体**和**翻越掩体**，从《全境封锁》《Outriders》和《Gears 5》等掩体交互相关的游戏来看，作为此类游戏高频操作的进入掩体一般为A键，脱离掩体一般为B键。值得一提的是，《Gears 5》中角色翻滚、进入掩体、翻越掩体均用A键来实现（A键和B键相比，A键优先跳跃然后才是进入掩体，如《神秘海域4）。

## 2.2 角色战斗操控

射击游戏中，角色战斗操控主要分为以枪械为主的核心战斗操控、以技能和道具为主的战斗辅助操控，以及互动、标记、发送动作表情等其他操控。

#### 2.2.1 枪械操控

射击游戏中，枪械操控主要指围绕枪械展开的例如瞄准、射击、装弹、切换武器等基础操控。除此之外，在一些大逃杀游戏中，对于枪械的操控和互动更加注重细节并还原真实，因此游戏中一般会提供一些高级功能，如调整瞄具倍率、切换射击模式、检视和收起武器等，以增加玩家与枪械的互动和操作深度。

![](http://km.fp.ps.netease.com/file/652f52e4cad4c4b9ebda2e47702TlXZK05?fop=imageView/6/f/webp/q/50)

1. 一般射击游戏中的**开火、瞄准、装弹**等功能可以采用图示进行按键设置，基本可以符合大多数玩家的操作习惯。若游戏中瞄准模式为肩瞄，则一般肩瞄状态下按下L键来切换肩位。若角色可同时装备两件或两件以上枪械，则一般通过点击Y键进行武器切换。

![](http://km.fp.ps.netease.com/file/652f4f1d0a0e31d6c8a54cc5IjsK0Eah05?fop=imageView/6/f/webp/q/50)

2. 若游戏中可供玩家选择的**瞄准**模式只有一种，一般默认以按住LT键来触发（支持在系统设置中将触发方式改为点击）。若存在多种瞄准模式，则一般通过LT键的不同输入方式来触发，比如《PUBG》按住LT键为肩瞄，点击LT键为机瞄/开镜；《遗迹系列》中按住LT键为肩瞄，双击LT键不放为开镜。

![](http://km.fp.ps.netease.com/file/6523e5c0edab17a552ef41f56u60sHsN05?fop=imageView/6/f/webp/q/50)

3. 开镜状态下**调整瞄具倍率**时，若仅有两种倍率，可以像《Apex》一样通过点击某个按键来实现切换；若倍率选择较多，则《PUBG》通过摇杆前后拖动来放大缩小倍率更加符合玩家直觉。

4. **装弹/切换武器：**手柄的A键、B键用于角色基础行动，剩余的X键一般用于装弹，而Y键一般用来调整武器。单击Y键切换主武器，双击或长按Y键切换副武器或收起武器。切换武器时建议采用双击输入，效率更加且玩家感知清晰，更具有掌控感。

#### 2.2.2 使用道具和技能

根据游戏类型和风格不同，射击游戏中除使用枪械进行射击外，玩家还可以使用道具或技能进行战斗辅助，以此来提高玩家操作深度和游戏可玩性。

![](http://km.fp.ps.netease.com/file/652f65ae00b49ab8e4973b41BBR3bO2e05?fop=imageView/6/f/webp/q/50)

1.从上图可以看出，一般**方向键**用于道具选择和使用，**肩键**用于技能施放。大多数游戏中，技能相较于道具而言对于角色战斗影响程度更大或更能体现游戏核心战斗特色，由此可见手柄中肩键的使用频率是高于方向键的，当然根本原因还是因为手柄肩键处于默认手势下手指直接控制区域。《Otriders》作为一款以技能为重要输出的射击游戏，其肩键用于技能施放而方向键用于目标追踪、打开任务界面等低频使用功能。同样的《COD 战区2.0》以道具使用作为辅助战斗的重要输出，其肩键用于道具使用而方向键用于标记、切换射击模式、打开背包等低频使用功能。

2. 游戏中的道具一般分为医疗品和投掷物，一般通过同一个方向键的不同输入方式来触发道具使用和打开道具轮盘。除此之外，由于投掷物需要先选中再通过点击RT键来使用，所以一般都具有**点击按键进行切换选中投掷物**的功能。

3.一般情况下，**肩键组合键**为更强大技能或道具，如《APEX》中按住LB+RB为施放英雄大招，《COD 战区2.0》中按住LB+RB为装备更加强大的战场升级装备。图示《Outriders》中普通技能2采用组合键是为了映射界面技能icon布局关系。界面技能icon和手柄LB、RB两个按键位置关系统一，更加符合玩家对于按键功能映射的直觉。

#### 2.2.3 局内其他操作

这一部分内容主要指局内除角色战斗之外的其他操作，虽然对于射击游戏来讲对于战斗体验影响不大， 但是属于游戏必备功能，对于玩家在游戏内的决策和互动起到重要作用。

![](http://km.fp.ps.netease.com/file/652f4e3967c12ac2e0e56808pcZEFkDm05?fop=imageView/6/f/webp/q/50)

**互动**：互动键和装弹键的一般都用X，通过输入方式来进行功能区分。可以从下表中看出：大多数射击游戏为了确保装弹的效率和准确性，大多采用点击X装弹长按X互动的方式。但是诸如《PUBG》《APEX》这类大逃杀玩法的游戏，为了确保捡物资的效率而采用点击X互动。《PUBG》因此不得不的长按X设置为装弹而牺牲装弹的效率。可以看出在《APEX》和《遗迹2》中装弹和互动均统一采用点击X来实现，需要注意的是《APEX》中玩家对于点击X是装弹还是互动具有清晰的感知，而《遗迹2》中背包无容量限制且不存在垃圾道具，就算误操作也不会对玩家造成损失。

![](http://km.fp.ps.netease.com/file/652e7f258c4f434d95136753higZJmZH05?fop=imageView/6/f/webp/q/50)

**近战**：常规射击游戏中近战都作为一种低频出现的功能，所以采用按下右摇杆来实现。当近战功能更加重要时，可以考虑采用扳机键或字母键，比如《遗迹2》中采用RT键近战攻击，《Warframe》和《Gears 5》均采用点击B键近战攻击。

**标记**和标记轮盘，**表情**和表情轮盘均采用单击和长按的输入方式来实现，减少玩家对于键位映射的记忆压力。

**自由视角**：需要同时操作左右两个摇杆，此时从操作舒适度角度考虑，只能通过按下某个肩键来实现。

## 2.3 全局操控

这一项内容主要指对游戏全局通用性操作

![](http://km.fp.ps.netease.com/file/652e7b8252ffc88dbf11287eOhzxGiib05?fop=imageView/6/f/webp/q/50)

1. **肩键**和**扳机键**一般用于对导航页签进行切换，LB/RB控制一级页签，LT/RT控制二级页签，其在界面上的位置关系和手柄按键位置关系完全契合，减少玩家记忆键位的压力。

2. 手柄其余两个键：BACK键和START键。**BACK键**一般用来打开和关闭地图界面，**START键**一般用来打开和关闭背包/游戏菜单界面。大逃杀游戏中因为打开背包界面属于高频操作，所以点击按键打开背包，长按按键打开系统菜单。

## 2.4 地图界面

左主机游戏中， 无论是焦点式还是光标式，在打开地图界面时均会显示光标。所以地图界面作为一个特殊功能界面还是需要单独拿出来介绍的。

![](http://km.fp.ps.netease.com/file/652e7d97154c2740bd6dc832VMivDgq905?fop=imageView/6/f/webp/q/50)

1. 地图界面是移动光标还是移动地图，不同游戏有不同的做法。一般可以一屏展示的地图界面均采用移动光标的方式，若地图较大需要满足当光标移动至屏幕边缘时触发地图移动。若采用移动地图的方式，因为光标始终处于屏幕中央，所以需要地图边缘有足够的空白区域，致使地图界面中所有的图例都可以被光标选中。

2. 若游戏场景也具有标记功能，比如大逃杀游戏，那么为了按键功能统一，在地图界面里仍采用该键位进行地图界面标记。

# 三. 手柄按键映射放方法和原则

## 3.1 手柄按键映射方法

在对3D射击游戏进行手柄按键映射时，首先要对游戏中玩家所有可进行的操作进行汇总、分类和重要级划分，再根据操作重要级进行按键分配，最后跟进落地并不断体验和调优。

![](http://km.fp.ps.netease.com/file/653a066e0b1b2261f24d429axGoO39N305?fop=imageView/6/f/webp/q/50)

需要注意的是，在对游戏操作进行重要级划分时，需要考虑角色当前处所的状态。比如当角色处于驾驶状态时，加速、刹车和转向操作是核心操作；当角色处于游泳状态时，向上游、向下游和加速是核心操作。

## 3.2 手柄按键映射原则

#### **3.2.1 遵循“国际惯例”**

尽管射击游戏有多种模式和玩法，但围绕着枪械的核心操作是固定的，这些操作所对应的输入按键在各个射击游戏中也是保持一致的，并已经形成了“国际惯例”。因此，在进行手柄按键映射时，按照“国际惯例”设置通用操作按键，可以最大程度地符合玩家的认知和操作习惯。

![](http://km.fp.ps.netease.com/file/653f9e8f137d30ca818e257b6ilKeHG905?fop=imageView/6/f/webp/q/50)

#### **3.2.2 高频操作优先分配高舒适度按键**

根据操作频率优先分配高舒适度按键，确保角色行动和战斗相关的基础操作都是可以轻松、快速的完成。并且，结合费茨定律确保手指长时间在多个按键之间频繁切换时不会导致肌肉酸痛。

#### **3.2.3 按键功能一致性**

确保同一按键在不同界面和操作状态下的功能一致，例如《APEX》战斗场景中点击RB键为场景标记，地图界面中点击RB键为地图标记。例如《PUBG》中拖动R键无论在默认状态还是组合键按下RB触发自由视角的状态下，均为控制视角的功能。最大程度避免按键功能错乱，减少玩家记忆负担。

#### **3.2.4 低操作压力**

通过按键设置降低操作压力主要从两方面考虑：（1）控制组合键数量在3个以内，避免同时操作4根手指；（2）左右手合理分配操作，避免一手多操和跨区操作。

#### **3.2.5 恰当运用一键多用**

在出现功能接近时可以使用一键多用，比如L键不同的输入方式可以实现不同操作，推动实现移动，按下后推动实现奔跑，双击实现自动奔跑。不仅可以使玩家高效切换移动状态，而且还可以减少操作过多时按键数量不足带来的压力。

#### **3.2.6 特殊按键和常见功能强绑定**

特殊按键如Map键，虽然在操作舒适度上低于方向键和功能键，但是应尽量避免使用空置的方向键或功能键来打开地图界面，这和玩家已经形成的手柄按键映射模型不符。

#### **3.2.7 契合自然映射**

涉及到游戏中某些关于方向或空间关系调整的操控，最好使对应按键的方位关系和其功能具有一定的相关性。或者使控件在游戏界面中的布局关系和对应按键在手柄上的布局相同，比如《遗迹：灰烬重生》中，通过手柄方向键来控制4种道具的使用。这4种道具在游戏HUD界面中的布局正好也具有上下左右的关系。当操控和反馈契合自然映射时，按键分配方案才是最容易被玩家接受和使用的。
