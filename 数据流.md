``` mermaid
sequenceDiagram
    autonumber
    actor User as 👤 User (用户)
    participant Memory as 🧠 Memory (上下文/历史)
    participant Assistant as 🤖 Assistant (AI Agent)
    participant Tool as 🛠️ Tool (工具/环境)

    %% 阶段1：输入
    User->>Assistant: 提交任务 (User Prompt)
    Assistant->>Memory: 1. 记录 User Message
    
    %% 阶段2：推理与执行循环 (Reasoning Loop)
    loop Agent Reasoning Loop (可迭代多次)
        Memory-->>Assistant: 2. 读取当前所有上下文 (包含可用工具列表)
        
        Note over Assistant: 3. 内部推理 (Thought): <br/>分析当前目标与已有信息。<br/>决定是调用工具还是输出最终答案。
        
        alt 需要调用工具 (Action)
            Assistant->>Memory: 4a. 记录 Assistant Message (包含 Tool Call 请求)
            Assistant->>Tool: 4b. 发送工具调用指令 (函数名 + JSON参数)
            activate Tool
            Note over Tool: 5. 执行具体的代码/API/搜索等物理动作
            Tool-->>Assistant: 6. 返回执行结果 (Observation / Tool Result)
            deactivate Tool
            Assistant->>Memory: 7. 记录 Tool Message (执行结果数据)
            
        else 任务已完成 (Finish)
            Note over Assistant: 8. 判断信息已充分，准备回答
            Assistant->>Memory: 9. 记录 Assistant Message (最终文本回复)
        end
    end

    %% 阶段3：输出
    Assistant-->>User: 10. 返回最终结果 (Final Response)
```
